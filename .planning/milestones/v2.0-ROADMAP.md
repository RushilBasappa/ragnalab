# Milestone v2.0: Network Services

**Status:** SHIPPED 2026-01-20
**Phases:** 5-7.1
**Total Plans:** 19

## Overview

v2.0 extended the homelab to provide network-wide services for the entire home network, starting with DNS-based ad blocking via Pi-hole, then adding a complete media automation stack, and finishing with operational hardening including folder restructure, socket-proxy security, Autokuma monitoring, and Backrest backup system.

## Phases

### Phase 5: Pi-hole Network-Wide Ad Blocking

**Goal:** Network-wide DNS-based ad blocking with Pi-hole as DHCP server and automatic fallback for high availability
**Depends on:** Phase 4 (existing infrastructure)
**Plans:** 3 plans

Plans:
- [x] 05-01-PLAN.md — Pi-hole Docker deployment with macvlan network and Traefik integration
- [x] 05-02-PLAN.md — DHCP configuration and network cutover from Xfinity gateway (*DNS-only mode - gateway locked*)
- [x] 05-03-PLAN.md — Monitoring, Homepage widget, backup, and full verification

**Details:**
- Macvlan networking for dedicated LAN IP (10.0.0.200)
- Macvlan-shim at 10.0.0.201 for host-to-container communication
- DNS-only mode due to locked Xfinity XB8 gateway (DHCP cannot be disabled)
- Manual device DNS configuration required

## Milestone Summary

**Bonus Deploys (outside GSD):**
- Plex media server
- Recyclarr (arr quality sync)
- Flaresolverr (indexer helper)
- Maintainerr (media cleanup)

**Key Decisions:**
- DNS-only mode for Pi-hole (Xfinity gateway locked)
- ProtonVPN WireGuard for torrent privacy
- Direct-play only for Jellyfin (Pi 5 no hardware encoding)
- Nested includes pattern for compose organization
- Socket-proxy for all Docker API access
- Autokuma labels for automatic monitoring

**Issues Resolved:**
- Macvlan host communication via shim interface
- network_mode: container:gluetun for VPN routing
- Jellyfin API bootstrap script complexity
- qBittorrent VPN binding with dynamic tun0 IP

**Issues Deferred:**
- Phase 8 apps moved to v3.0 (SSO-first approach)
- Pi-hole DHCP mode (requires gateway replacement)

**Technical Debt Incurred:**
- Phase 7.1 deployed manually without formal plans
- Some bootstrap scripts need refactoring
