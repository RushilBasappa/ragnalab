### Phase 11: New App Deployment

1. **Paperless health check** - Container shows "unhealthy" due to celery worker memory constraints on Raspberry Pi. Webserver responds correctly. Not a blocker but should be tuned.

2. **Session cookie domain** - Configured as `ragnalab.xyz` rather than `.ragnalab.xyz` (with leading dot). Modern browsers handle both, but explicit leading dot is more compatible per RFC 6265.
