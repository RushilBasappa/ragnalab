<task type="auto">
  <name>Task 1: Create Prowlarr docker-compose.yml</name>
  <files>apps/media/prowlarr/docker-compose.yml</files>
  <action>
Create apps/media/prowlarr/docker-compose.yml following the established pattern.

Configuration:
- Image: lscr.io/linuxserver/prowlarr:latest
- Container name: prowlarr
- Environment:
  - PUID=1000
  - PGID=1000
  - TZ=America/Los_Angeles
- Volumes:
  - prowlarr-config:/config
- Networks: proxy (external)
- Ports: None (Traefik handles routing)
- Labels:
  Traefik:
  - traefik.enable=true
  - traefik.http.routers.prowlarr.rule=Host(`prowlarr.ragnalab.xyz`)
  - traefik.http.routers.prowlarr.entrypoints=websecure
  - traefik.http.routers.prowlarr.tls=true
  - traefik.http.routers.prowlarr.tls.certresolver=letsencrypt
  - traefik.http.services.prowlarr.loadbalancer.server.port=9696
  - traefik.docker.network=proxy
  Homepage:
  - homepage.group=Media
  - homepage.name=Prowlarr
  - homepage.icon=prowlarr.png
  - homepage.href=https://prowlarr.ragnalab.xyz
  - homepage.description=Indexer Manager
  - homepage.widget.type=prowlarr
  - homepage.widget.url=http://prowlarr:9696
  - homepage.widget.key=${PROWLARR_API_KEY}
  - homepage.server=my-docker
  Backup:
  - docker-volume-backup.stop-during-backup=prowlarr
- Resource limits: memory 256M, cpus 0.25, reservation 128M
- Logging: json-file, max-size 10m, max-file 3
- restart: unless-stopped

Include volumes and networks sections at bottom.
  </action>
  <verify>cat apps/media/prowlarr/docker-compose.yml | grep -q "prowlarr.ragnalab.xyz"</verify>
  <done>Prowlarr docker-compose.yml created with Traefik and Homepage labels</done>

<task type="auto">
  <name>Task 2: Deploy Prowlarr and configure authentication</name>
  <files>apps/media/prowlarr/docker-compose.yml</files>
  <action>
1. Start Prowlarr:
   cd /home/rushil/workspace/ragnalab/apps/media/prowlarr
   docker compose up -d

2. Wait for container to be ready (15-20 seconds):
   docker logs prowlarr 2>&1 | tail -5

3. Verify Traefik discovers the route:
   curl -s http://localhost:8080/api/rawdata | grep -i prowlarr

4. Test HTTPS access:
   curl -ks https://prowlarr.ragnalab.xyz -o /dev/null -w "%{http_code}"

5. Access Prowlarr at https://prowlarr.ragnalab.xyz

6. On first access, Prowlarr will prompt for authentication setup:
   - Choose "Forms (Login Page)" for Authentication Method
   - Set Username and Password
   - This is important - Prowlarr has no default auth

7. After login, get API key for Homepage widget:
   Settings -> General -> Security -> API Key
   Copy this key.

8. Add to apps/media/.env:
   PROWLARR_API_KEY=your_api_key_here

9. Restart Prowlarr to pick up Homepage widget:
   docker compose down && docker compose --env-file ../.env up -d

Note: Do not add indexers yet - that happens after Sonarr/Radarr are deployed.
  </action>
  <verify>curl -ks https://prowlarr.ragnalab.xyz -o /dev/null -w "%{http_code}" | grep -q "200\|401\|302"</verify>
  <done>Prowlarr accessible at https://prowlarr.ragnalab.xyz with authentication configured</done>
</task>
