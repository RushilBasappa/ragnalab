<task type="auto">
  <name>Task 1: Create Jellyfin docker-compose.yml</name>
  <files>apps/media/jellyfin/docker-compose.yml</files>
  <action>
Create apps/media/jellyfin/docker-compose.yml.

IMPORTANT: Pi 5 cannot hardware transcode. Configure for direct-play only.

Configuration:
- Image: lscr.io/linuxserver/jellyfin:latest
- Container name: jellyfin
- Environment:
  - PUID=1000
  - PGID=1000
  - TZ=America/Los_Angeles
  - JELLYFIN_PublishedServerUrl=https://jellyfin.ragnalab.xyz
- Volumes:
  - jellyfin-config:/config
  - /media/library:/data/media:ro (read-only - Jellyfin doesn't modify media)
- Networks: proxy (external)
- Labels:
  Traefik:
  - traefik.enable=true
  - traefik.http.routers.jellyfin.rule=Host(`jellyfin.ragnalab.xyz`)
  - traefik.http.routers.jellyfin.entrypoints=websecure
  - traefik.http.routers.jellyfin.tls=true
  - traefik.http.routers.jellyfin.tls.certresolver=letsencrypt
  - traefik.http.services.jellyfin.loadbalancer.server.port=8096
  - traefik.docker.network=proxy
  Homepage:
  - homepage.group=Media
  - homepage.name=Jellyfin
  - homepage.icon=jellyfin.png
  - homepage.href=https://jellyfin.ragnalab.xyz
  - homepage.description=Media Server
  - homepage.widget.type=jellyfin
  - homepage.widget.url=http://jellyfin:8096
  - homepage.widget.key=${JELLYFIN_API_KEY}
  - homepage.server=my-docker
  Backup:
  - docker-volume-backup.stop-during-backup=jellyfin
- Resource limits: memory 1G, cpus 1.0, reservation 512M (larger for media serving)
- Logging: json-file, max-size 10m, max-file 3
- restart: unless-stopped

DO NOT include V4L2 device mounts - V4L2 is deprecated by Jellyfin and provides no benefit for direct-play.

Include volumes and networks sections.
  </action>
  <verify>cat apps/media/jellyfin/docker-compose.yml | grep -q "jellyfin.ragnalab.xyz"</verify>
  <done>Jellyfin docker-compose.yml created with direct-play configuration</done>
</task>

<task type="auto">
  <name>Task 2: Deploy Jellyfin and complete initial setup</name>
  <files>apps/media/jellyfin/docker-compose.yml</files>
  <action>
1. Start Jellyfin:
   cd /home/rushil/workspace/ragnalab/apps/media/jellyfin
   docker compose up -d

2. Wait for startup (30-60 seconds, first start downloads metadata providers):
   docker logs jellyfin 2>&1 | tail -10

3. Verify Traefik route:
   curl -ks https://jellyfin.ragnalab.xyz -o /dev/null -w "%{http_code}"

4. Access Jellyfin at https://jellyfin.ragnalab.xyz

5. Complete Setup Wizard:
   a. Select preferred language
   b. Create admin user (username/password)
   c. Add Media Libraries:
      - Movies: /data/media/movies
      - TV Shows: /data/media/tv
   d. Configure Metadata settings (defaults are fine)
   e. Configure Remote Access (Traefik handles this, defaults okay)
   f. Finish wizard

6. Get API key for Homepage widget:
   Dashboard -> API Keys -> Add (name: "homepage")
   Add to apps/media/.env: JELLYFIN_API_KEY=your_key

7. Disable Transcoding (CRITICAL for Pi 5):
   Dashboard -> Playback -> Transcoding
   - Hardware acceleration: None
   - Enable hardware decoding: OFF
   - All transcoding options: Disabled/Off
   This forces clients to direct-play, which is what Pi 5 can handle.

8. Restart with API key:
   docker compose --env-file ../.env up -d
  </action>
  <verify>curl -ks https://jellyfin.ragnalab.xyz -o /dev/null -w "%{http_code}" | grep -q "200\|302"</verify>
  <done>Jellyfin accessible with libraries configured and transcoding disabled</done>
</task>
