  <name>Task 1: Verify all containers running and healthy</name>
  <files>N/A</files>
  <action>
Run comprehensive health check on all media stack containers.

1. List all media containers:
   docker ps --filter "name=gluetun" --filter "name=qbittorrent" --filter "name=prowlarr" \
     --filter "name=sonarr" --filter "name=radarr" --filter "name=bazarr" \
     --filter "name=unpackerr" --filter "name=jellyfin" --filter "name=jellyseerr" \
     --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"

2. Verify Gluetun is healthy:
   docker inspect gluetun --format='{{.State.Health.Status}}'
   Expected: healthy

3. Verify VPN routing:
   docker exec gluetun wget -qO- ifconfig.me
   docker exec qbittorrent curl -s ifconfig.me
   Both should show same VPN IP (NOT home IP)

4. Check container logs for errors:
   for c in gluetun qbittorrent prowlarr sonarr radarr bazarr unpackerr jellyfin jellyseerr; do
     echo "=== $c ==="
     docker logs $c 2>&1 | tail -5 | grep -i error || echo "No errors"
   done

5. Verify Traefik routes exist:
   curl -s http://localhost:8080/api/rawdata | jq '.routers | keys[]' | grep -E "prowlarr|sonarr|radarr|bazarr|jellyfin|jellyseerr"

Expected: 9 containers running (gluetun, qbittorrent, prowlarr, sonarr, radarr, bazarr, unpackerr, jellyfin, jellyseerr)
  </action>
  <verify>docker ps --format '{{.Names}}' | grep -cE "gluetun|qbittorrent|prowlarr|sonarr|radarr|bazarr|unpackerr|jellyfin|jellyseerr"</verify>
  <done>All 9 media containers running with no critical errors</done>

  <what-needed>Add indexers to Prowlarr</what-needed>
  <instructions>
Prowlarr needs indexers (torrent sites) to search for media. This requires manual configuration as it involves selecting specific indexers and potentially entering credentials.

1. Go to https://prowlarr.ragnalab.xyz

2. Navigate to: Indexers -> Add Indexer

3. Add public indexers (no account needed):
   - 1337x
   - EZTV (for TV)
   - The Pirate Bay
   - RARBG (if available)
   - YTS (for movies)

4. Add private indexers (if you have accounts):
   - TorrentLeech
   - IPTorrents
   - etc.

5. For each indexer:
   - Click Add
   - Configure any required settings
   - Test the connection
   - Save

6. Verify sync to Sonarr/Radarr:
   - Go to Settings -> Apps
   - Sonarr and Radarr should show "Sync in progress" or green checkmark
   - Indexers will appear in Sonarr/Radarr -> Settings -> Indexers

Note: Without indexers, Sonarr/Radarr cannot search for media.
  </instructions>
  <resume-signal>Type "indexers configured" when at least 2 indexers are added and synced</resume-signal>
</task>
