<task type="auto">
  <name>Task 2: Create Unpackerr docker-compose.yml</name>
  <files>apps/media/unpackerr/docker-compose.yml</files>
  <action>
Create apps/media/unpackerr/docker-compose.yml.

Unpackerr is a headless service (no web UI). It monitors download folders and extracts archives when arr apps import them.

Configuration:
- Image: hotio/unpackerr:latest
- Container name: unpackerr
- Environment:
  - PUID=1000
  - PGID=1000
  - TZ=America/Los_Angeles
  # Sonarr connection
  - UN_SONARR_0_URL=http://sonarr:8989
  - UN_SONARR_0_API_KEY=${SONARR_API_KEY}
  - UN_SONARR_0_PATHS_0=/media/downloads/tv
  # Radarr connection
  - UN_RADARR_0_URL=http://radarr:7878
  - UN_RADARR_0_API_KEY=${RADARR_API_KEY}
  - UN_RADARR_0_PATHS_0=/media/downloads/movies
- Volumes:
  - /media:/media (needs write access for extraction)
- Networks: default (media-internal network for arr communication)
- Resource limits: memory 128M, cpus 0.25
- Logging: json-file, max-size 10m, max-file 3
- restart: unless-stopped

Note: hotio/unpackerr uses PUID/PGID environment variables (matches LinuxServer pattern), unlike golift/unpackerr which uses --user flag.

Create or reference a shared network for arr apps:
```yaml
networks:
  default:
    name: media-internal
```
  </action>
  <verify>cat apps/media/unpackerr/docker-compose.yml | grep -q "UN_SONARR_0_URL"</verify>
  <done>Unpackerr docker-compose.yml created with Sonarr and Radarr connections</done>
</task>

<task type="auto">
  <name>Task 3: Deploy and configure Bazarr</name>
  <files>apps/media/bazarr/docker-compose.yml</files>
  <action>
1. Start Bazarr:
   cd /home/rushil/workspace/ragnalab/apps/media/bazarr
   docker compose up -d

2. Wait for startup:
   docker logs bazarr 2>&1 | tail -5

3. Verify Traefik route:
   curl -ks https://bazarr.ragnalab.xyz -o /dev/null -w "%{http_code}"

4. Access Bazarr at https://bazarr.ragnalab.xyz

5. Configure authentication:
   Settings -> General -> Security
   - Authentication: Forms
   - Set Username and Password

6. Get API key:
   Settings -> General -> API Key
   Add to apps/media/.env: BAZARR_API_KEY=your_key

7. Connect to Sonarr:
   Settings -> Sonarr
   - Address: sonarr
   - Port: 8989
   - API Key: (from apps/media/.env SONARR_API_KEY)
   - Test and Enable

8. Connect to Radarr:
   Settings -> Radarr
   - Address: radarr
   - Port: 7878
   - API Key: (from apps/media/.env RADARR_API_KEY)
   - Test and Enable

9. Configure Subtitle Providers:
   Settings -> Providers -> Add
   - OpenSubtitles.org (requires free account)
   - Subscene
   - Add any other preferred providers

10. Set Languages:
    Settings -> Languages
    - Add preferred languages for subtitles
    - Set default profile

11. Restart with API key:
    docker compose --env-file ../.env up -d
  </action>
  <verify>curl -ks https://bazarr.ragnalab.xyz -o /dev/null -w "%{http_code}" | grep -q "200\|401\|302"</verify>
  <done>Bazarr accessible and connected to Sonarr/Radarr</done>
</task>
