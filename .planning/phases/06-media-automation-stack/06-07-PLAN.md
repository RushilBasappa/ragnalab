  <name>Task 1: Create Jellyseerr docker-compose.yml</name>
  <files>apps/media/jellyseerr/docker-compose.yml</files>
  <action>
Create apps/media/jellyseerr/docker-compose.yml.

Configuration:
- Image: fallenbagel/jellyseerr:latest
- Container name: jellyseerr
- Environment:
  - TZ=America/Los_Angeles
  - LOG_LEVEL=info
  (Note: Jellyseerr doesn't use PUID/PGID - it runs as node user internally)
- Volumes:
  - jellyseerr-config:/app/config
- Networks: proxy (external)
- Labels:
  Traefik:
  - traefik.enable=true
  - traefik.http.routers.jellyseerr.rule=Host(`requests.ragnalab.xyz`)
  - traefik.http.routers.jellyseerr.entrypoints=websecure
  - traefik.http.routers.jellyseerr.tls=true
  - traefik.http.routers.jellyseerr.tls.certresolver=letsencrypt
  - traefik.http.services.jellyseerr.loadbalancer.server.port=5055
  - traefik.docker.network=proxy
  Homepage:
  - homepage.group=Media
  - homepage.name=Jellyseerr
  - homepage.icon=jellyseerr.png
  - homepage.href=https://requests.ragnalab.xyz
  - homepage.description=Media Requests
  - homepage.widget.type=jellyseerr
  - homepage.widget.url=http://jellyseerr:5055
  - homepage.widget.key=${JELLYSEERR_API_KEY}
  - homepage.server=my-docker
  Backup:
  - docker-volume-backup.stop-during-backup=jellyseerr
- Resource limits: memory 512M, cpus 0.5, reservation 256M
- Logging: json-file, max-size 10m, max-file 3
- restart: unless-stopped

Include volumes and networks sections.
  </action>
  <verify>cat apps/media/jellyseerr/docker-compose.yml | grep -q "requests.ragnalab.xyz"</verify>

<task type="auto">
  <name>Task 2: Deploy Jellyseerr and configure integrations</name>
  <files>apps/media/jellyseerr/docker-compose.yml</files>
  <action>
1. Start Jellyseerr:
   cd /home/rushil/workspace/ragnalab/apps/media/jellyseerr
   docker compose up -d

2. Wait for startup:
   docker logs jellyseerr 2>&1 | tail -10

3. Verify Traefik route:
   curl -ks https://requests.ragnalab.xyz -o /dev/null -w "%{http_code}"

4. Access Jellyseerr at https://requests.ragnalab.xyz

5. Complete Setup Wizard - Jellyfin Integration:
   a. Select "Use your Jellyfin account"
   b. Jellyfin URL: http://jellyfin:8096
   c. Email: your admin email
   d. Username: your Jellyfin admin username
   e. Password: your Jellyfin admin password
   f. Complete sign-in

6. Configure Media Server:
   Settings -> Jellyfin
   - Verify connection is working
   - Enable "Sync libraries" for Movies and TV Shows

7. Add Radarr:
   Settings -> Services -> Radarr -> Add
   - Default Server: Yes
   - Server Name: Radarr
   - Hostname: radarr
   - Port: 7878
   - API Key: (from apps/media/.env RADARR_API_KEY)
   - Test and Save

8. Add Sonarr:
   Settings -> Services -> Sonarr -> Add
   - Default Server: Yes
   - Server Name: Sonarr
   - Hostname: sonarr
   - Port: 8989
   - API Key: (from apps/media/.env SONARR_API_KEY)
   - Test and Save

9. Get API key for Homepage:
   Settings -> General -> API Key
   Add to apps/media/.env: JELLYSEERR_API_KEY=your_key

10. Restart with API key:
    docker compose --env-file ../.env up -d
  </action>
  <verify>curl -ks https://requests.ragnalab.xyz -o /dev/null -w "%{http_code}" | grep -q "200\|302"</verify>
  <done>Jellyseerr accessible and connected to Jellyfin, Sonarr, and Radarr</done>
</task>
