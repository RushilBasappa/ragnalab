<objective>
Install Tailscale on the Raspberry Pi host for remote VPN access

Purpose: Enable remote access to homelab services via Tailscale VPN while keeping local network access available. Host-level install is simpler and more robust than containerized Tailscale.

Output: Pi accessible via Tailscale for remote access, local access unchanged
</objective>

<context>
@.planning/PROJECT.md
@.planning/ROADMAP.md

</context>

<tasks>

<task type="auto">
  <name>Task 1: Install Tailscale on host</name>
  <files>None (system package)</files>
  <action>
Install Tailscale using the official install script:

```bash
# Install Tailscale
curl -fsSL https://tailscale.com/install.sh | sh

# Verify installation
tailscale version
```

This installs Tailscale as a system service that starts on boot.
  </action>
  <verify>
Run: `tailscale version`
Expected: Shows Tailscale version (e.g., 1.x.x)
  </verify>
  <done>Tailscale installed on host</done>
</task>

<task type="checkpoint:human-action" gate="blocking">
  <action>Authenticate Tailscale to your tailnet</action>
  <instructions>
Run the following command and follow the authentication URL:

```bash
sudo tailscale up
```

1. A URL will be printed - open it in your browser
2. Log in to your Tailscale account
3. Authorize the device
4. The command will complete once authenticated

After authentication, verify with:
```bash
tailscale status
tailscale ip -4
```
  </instructions>
  <resume-signal>Type "done" after Tailscale is authenticated</resume-signal>
</task>

<task type="auto">
  <name>Task 2: Verify Tailscale connectivity</name>
  <files>None (verification)</files>
  <action>
Verify Tailscale is working and get the assigned IP:

```bash
# Check Tailscale status
tailscale status

# Get Tailscale IPv4 address
TS_IP=$(tailscale ip -4)
echo "Tailscale IP: $TS_IP"

# Verify the tailscale0 interface exists
ip addr show tailscale0
```
  </action>
  <verify>
Run: `tailscale status`
Expected: Shows "ragnalab" (or hostname) as connected to tailnet
  </verify>
  <done>Tailscale connected to tailnet</done>
</task>

<task type="auto">
  <name>Task 3: Verify dual access (local + VPN)</name>
  <files>None (verification)</files>
  <action>
Verify services are accessible via both local network and Tailscale IP:

```bash
# Get IPs
TS_IP=$(tailscale ip -4)
LOCAL_IP=$(hostname -I | awk '{print $1}')

echo "Tailscale IP: $TS_IP"
echo "Local IP: $LOCAL_IP"

# Test via Local IP (should work)
echo "Testing via Local IP..."
curl -sk --connect-timeout 5 "https://$LOCAL_IP" -H "Host: whoami.ragnalab.xyz" | head -3

# Test via Tailscale IP (should work)
echo "Testing via Tailscale IP..."
curl -sk --connect-timeout 5 "https://$TS_IP" -H "Host: whoami.ragnalab.xyz" | head -3
```

Both should return the whoami response.
  </action>
  <verify>
Both Local IP and Tailscale IP return whoami response
  </verify>
  <done>Services accessible via both local network and Tailscale VPN</done>
</task>

</tasks>
