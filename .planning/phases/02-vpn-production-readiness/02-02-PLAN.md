<task type="checkpoint:human-action" gate="blocking">
  <action>Create Tailscale OAuth client for container authentication</action>
  <instructions>
1. Go to https://login.tailscale.com/admin/settings/oauth
2. Click "Generate OAuth client"
3. Set scope to "devices:write"
4. Optionally add tags like "tag:server" (must exist in ACL policy)
5. Click "Generate" and copy the client secret (starts with tskey-client-...)
6. The secret will be used as TS_AUTHKEY environment variable

Note: OAuth client secrets never expire, unlike auth keys which max out at 90 days.
  </instructions>
  <resume-signal>Paste the OAuth client secret (tskey-client-...) or type "done" if already configured</resume-signal>
</task>

<task type="auto">
  <name>Task 2: Add Tailscale container and modify Traefik networking</name>
  <files>proxy/docker-compose.yml</files>
  <action>
Modify proxy/docker-compose.yml to:

1. Add Tailscale container as network gateway
2. Change Traefik to use `network_mode: service:tailscale`
3. Move port bindings from Traefik to Tailscale container
4. Ensure Tailscale is on both proxy and socket_proxy_network
5. Add tailscale_state volume for persistence

Critical changes:
- Tailscale container gets ports 80 and 443
- Traefik loses its ports and networks sections (inherits from Tailscale)
- Tailscale must be on socket_proxy_network so Traefik can reach socket-proxy
- Add TS_ROUTES for Docker proxy network subnet (172.22.0.0/16 or actual subnet)

Get actual Docker proxy network subnet:
```bash
docker network inspect proxy --format '{{range .IPAM.Config}}{{.Subnet}}{{end}}'
```

Use complete docker-compose.yml from 02-RESEARCH.md Pattern 1 as reference.

Key Tailscale environment variables:
- TS_AUTHKEY=${TS_AUTHKEY}
- TS_EXTRA_ARGS=--advertise-tags=tag:server --accept-routes
- TS_STATE_DIR=/var/lib/tailscale
- TS_USERSPACE=false
- TS_AUTH_ONCE=true
- TS_ROUTES=<docker_proxy_subnet>

Add restart: unless-stopped to all services for OPS-04.
  </action>
  <verify>
Run: `docker compose -f proxy/docker-compose.yml config 2>&1 | head -50`
Expected: Valid YAML, shows tailscale service with ports, traefik with network_mode
  </verify>
  <done>Docker Compose updated with Tailscale as network gateway for Traefik</done>
</task>

Verify services are only accessible via Tailscale, not local network:

```bash

LOCAL_IP=$(hostname -I | awk '{print $1}')
echo "Local IP: $LOCAL_IP"

# Test via Tailscale IP (should work)
echo "Testing via Tailscale IP..."

# Test via local IP (should fail/timeout)
echo "Testing via Local IP (should timeout/fail)..."

```
