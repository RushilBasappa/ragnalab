<objective>
Configure Sonarr, Radarr, and Prowlarr to use External authentication mode and add Authelia middleware.

Purpose: The *arr apps have their own Forms-based authentication. Setting AuthenticationMethod to "External" tells them to trust the Remote-User header from Authelia instead of showing their own login page. This eliminates double login while maintaining security.

Output: Three docker-compose.yml files updated with authelia@file middleware. All three *arr apps configured with External auth mode in their config.xml files.
</objective>

<tasks>

<task type="auto">
  <name>Task 1: Configure Prowlarr External auth and add middleware</name>
  <files>stack/media/prowlarr/docker-compose.yml</files>
  <action>
1. Stop the Prowlarr container:
   ```bash
   docker stop prowlarr
   ```

2. Edit Prowlarr's config.xml to set External authentication:
   - Config location: Docker volume `ragnalab_prowlarr-config` at `/config/config.xml`
   - Find the `<AuthenticationMethod>` element (likely set to "Forms" or "Basic")
   - Change it to: `<AuthenticationMethod>External</AuthenticationMethod>`
   - IMPORTANT: Ensure only ONE `<AuthenticationMethod>` element exists (remove duplicates)

   ```bash
   # Get the volume mount path
   docker volume inspect ragnalab_prowlarr-config --format '{{.Mountpoint}}'

   # Edit the config (as root since it's a Docker volume)
   sudo sed -i 's/<AuthenticationMethod>[^<]*<\/AuthenticationMethod>/<AuthenticationMethod>External<\/AuthenticationMethod>/' /var/lib/docker/volumes/ragnalab_prowlarr-config/_data/config.xml

   # Verify the change
   sudo grep "AuthenticationMethod" /var/lib/docker/volumes/ragnalab_prowlarr-config/_data/config.xml
   ```

3. Add Authelia middleware label to docker-compose.yml:
   Add this line in the labels section after the loadbalancer.server.port line:
   ```yaml
   - "traefik.http.routers.prowlarr.middlewares=authelia@file"
   ```

4. Start Prowlarr:
   ```bash
   docker start prowlarr
   ```
  </action>
  <verify>
```bash
# Verify config.xml has External auth
sudo grep "AuthenticationMethod" /var/lib/docker/volumes/ragnalab_prowlarr-config/_data/config.xml
# Expected: <AuthenticationMethod>External</AuthenticationMethod>

# Verify middleware label in docker-compose
grep "middlewares=authelia" /home/rushil/workspace/ragnalab/stack/media/prowlarr/docker-compose.yml
```
  </verify>
  <done>Prowlarr config.xml has External auth mode and docker-compose.yml has authelia middleware</done>
</task>
